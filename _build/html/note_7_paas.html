Heroku¶

Git用に公開鍵を準備する

cd C:Usersユーザ名 mkdir .ssh cd C:Program FilesGitbin ssh-keygen -t rsa

Herokuのアカウント作成

http://api.heroku.com/

Gemをインストール

gem install heroku heroku keys:add

登録する

git init git add . git commit -a -m “app init”

heroku create heroku addons:add sendgrid:starter git push heroku master heroku rake db:migrate heroku rake db:fixtures:load

動作確認

http://api.heroku.com/myapps
